# Bitcoin Cryptocurrency Data Analysis

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on historical Bitcoin market data to understand price trends, trading behaviour, and volatility patterns in the cryptocurrency market.

The analysis explores how Bitcoin prices have changed over time and examines relationships between price indicators such as **Open, High, Low, Close, and trading volume**.

---

## Dataset

The dataset used in this project is available on Kaggle:

Dataset Link:
https://www.kaggle.com/datasets/mczielinski/bitcoin-historical-data

This dataset contains **historical Bitcoin trading data at 1-minute intervals from multiple exchanges starting from January 2012**. ([Kaggle][1])

### Dataset Features

The dataset includes the following columns:

* **Timestamp** – UNIX timestamp of the record
* **Open** – Opening price of Bitcoin (USD)
* **High** – Highest price during the interval
* **Low** – Lowest price during the interval
* **Close** – Closing price of the interval
* **Volume_(BTC)** – Volume traded in Bitcoin
* **Volume_(Currency)** – Volume traded in USD
* **Weighted_Price** – Weighted average price
